💡 Cos'è un Programma?
Immagina di dare istruzioni dettagliate a un robot per fare qualcosa. Ogni passo deve essere specificato chiaramente: "primo, prendi il libro", "secondo, apri la pagina", "terzo, leggi il testo". Un programma è esattamente questo: una sequenza di istruzioni che il computer seguirà passo dopo passo.
🔤 Il Linguaggio della Macchina
Per comunicare con il computer, usiamo linguaggi come Python, Java o C#. Sono come lingue straniere: il computer capisce solo queste "lingue". Se gli dici "ciao" in italiano, non capisce; ma se usi le parole giuste in Python, allora reagisce!
💭 Ricorda: Programmazione = Comunicazione con il computer

📦 Variabili: Scatole di Memoria

🎁 L'Analogia della Scatola
Immagina il computer come una casa enorme con infinite scatole. Ogni scatola può contenere qualcosa: un numero, un nome, una risposta sì/no. Una "variabile" è semplicemente una scatola con un'etichetta. L'etichetta è il nome della variabile (identificatore), e quello che c'è dentro è il valore.
🏷️ Dare un Nome alla Scatola
Come quando etichetti una scatola con "Giocattoli" o "Libri", in programmazione dai nomi significativi alle variabili: nomeStudente, altezza, punteggio. Questo rende il codice facile da capire!

🎮 Crea le Tue Variabili

Assegna valori a variabili e vedi cosa succede:

Esempi di Variabili in Python:
nomeStudente = "Maria"
altezza = 170
eta = 16
èMaggiorenne = True

Tipi di Dati che le Variabili Possono Contenere:

🔢

Numeri (Integer)

Numeri interi come 170, 25, -5
eta = 16

📝

Testo (String)

Stringhe di testo come nomi
nome = "Marco"

✔️

Vero/Falso (Boolean)

Solo due opzioni: vero o falso
è_studente = True

🎯

Decimali (Float)

Numeri con virgola
altezza = 1.75

💡 Consiglio: Dai sempre nomi chiari alle variabili! Usa eta invece di x. Il codice futuro ti ringrazierà!

⚡ Costanti: Valori Immobili

🔒 L'Analogia della Scatola Bloccata
Se le variabili sono scatole normali dove puoi mettere cose diverse, le costanti sono scatole bloccate. Metti qualcosa dentro una volta, la chiudi e nessuno (neanche il programma) può cambiarla. Ad esempio, il valore di π (pi greco) è sempre 3.14159... non cambia mai!
📌 Quando Usare le Costanti?
Usa costanti per valori che non devono mai cambiare: il numero di giorni in una settimana (7), il valore di π, la velocità della luce, il nome della tua scuola, ecc.

🎮 Differenza tra Variabile e Costante

Costanti in Python (per convenzione):
PI_GRECO = 3.14159
GIORNI_DELLA_SETTIMANA = 7
VELOCITA_LUCE = 299792458 # m/s
NOME_SCUOLA = "Istituto Tecnico"

🗺️ Flowchart: La Mappa dell'Algoritmo

🧭 Che cosa è un Flowchart?
Un flowchart è come una mappa stradale per il computer. Invece di usando le parole (codice), usiamo disegni e simboli per mostrare come il programma deve funzionare. È come disegnare le strade e le curve su una mappa prima di guidare!

I Simboli del Flowchart

⭕

INIZIO / FINE

Ovale: segna dove il programma inizia e finisce

▭

AZIONE

Rettangolo: un'istruzione da eseguire (es: calcola, elabora)

◇

DECISIONE

Rombo: una domanda sì/no. Il flusso cambia in base alla risposta

▱

INPUT/OUTPUT

Parallelogramma: leggi dati (input) o stampa risultati (output)

➜

FRECCIA

Mostra la direzione del flusso, come una strada

Esempio: Calcolare la Media di 3 Numeri

Inizio
📥 INPUT: v1, v2, v3
🧮 media = (v1+v2+v3)/3
📤 OUTPUT: media
Fine
Codice Python corrispondente:
# Input
v1 = int(input("Primo numero: "))
v2 = int(input("Secondo numero: "))
v3 = int(input("Terzo numero: "))

# Azione (calcolo)
media = (v1 + v2 + v3) / 3

# Output
print("La media è: " + str(media))

Esempio con DECISIONE: Numero Positivo o Negativo?

Inizio
📥 INPUT: numero
numero > 0?
V
📤 OUTPUT:
"È positivo"
F
📤 OUTPUT:
"È negativo"

numero = int(input("Inserisci un numero: "))

💡 Ricorda: Il flowchart viene PRIMA del codice! È come fare la bozza di un disegno prima di colorarlo. Aiuta a capire la logica!

🔗 Algoritmi: Ricette per il Computer

👨‍🍳 L'Analogia della Ricetta
Un algoritmo è come una ricetta per una torta. La ricetta ti dice: "Primo, mescola la farina e le uova. Secondo, aggiungi il burro fuso. Terzo, cuoci a 180°C per 30 minuti." Un algoritmo fa lo stesso: dice al computer esattamente cosa fare, passo dopo passo, in ordine preciso.
📋 I Tre Passaggi Fondamentali
1. Analisi del Problema: Capisco cosa devo fare?
2. Creazione Algoritmo: Scrivo i passi per risolverlo (spesso su flowchart)
3. Codifica: Converto i passi in codice Python/Java/C#

Esempio Completo: Calcolare il Quadrato di un Numero

✅ PASSO 1: Analizzare il Problema

Cosa devo fare? Leggere un numero e stampare il suo quadrato.

Input: Un numero (es: 5)

Output: Il quadrato (es: 25)

📝 PASSO 2: Creare l'Algoritmo

Passi sequenziali:

1️⃣ Leggi il numero dall'utente

2️⃣ Moltiplica il numero per se stesso

3️⃣ Stampa il risultato

💻 PASSO 3: Codificare in Python
# Leggi il numero
numero = int(input("Inserisci un numero: "))

# Calcola il quadrato
quadrato = numero * numero

# Stampa il risultato
print("Il quadrato di " + str(numero) + " è " + str(quadrato))

I 3 Schemi Fondamentali degli Algoritmi

📊

SEQUENZA

Le azioni si eseguono una dopo l'altra in ordine
Es: leggi → calcola → stampa

🔀

SELEZIONE

Scegli un percorso in base a una condizione
Es: se numero > 0 stampa "positivo"

🔁

RIPETIZIONE

Ripeti un'azione più volte
Es: conta da 1 a 10

💡 Consiglio Finale: Scrivi sempre l'algoritmo PRIMA di scrivere il codice! Questo ti aiuta a pensare chiaramente e a evitare errori.
